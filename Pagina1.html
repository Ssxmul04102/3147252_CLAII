Glosario TI (N - Z)

N

Node.js: Entorno de ejecución para JavaScript en el lado del servidor.

O

Open Source: Software cuyo código fuente está disponible para ser modificado y distribuido libremente.

P

Python: Lenguaje de programación popular por su simplicidad y versatilidad.

Q

Query: Consulta realizada a una base de datos.

R

REST: Estilo arquitectónico para diseñar servicios web.

S

SQL: Lenguaje utilizado para gestionar bases de datos relacionales.

T

TCP/IP: Conjunto de protocolos que permiten la comunicación en redes.

U

UI: Interfaz de usuario, parte visual de una aplicación.

V

VPN: Red privada virtual que permite conexiones seguras.

W

WebSocket: Protocolo para comunicación bidireccional en tiempo real.

X

XML: Lenguaje de marcado utilizado para almacenar y transportar datos.

Y

YAML: Formato de serialización de datos legible por humanos.

Z

Zero-Day: Vulnerabilidad desconocida en software que puede ser explotada.
