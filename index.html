Glosario TI (A - M)

A

API: Interfaz de programación de aplicaciones que permite la comunicación entre diferentes sistemas.

B

Backend: Parte del desarrollo web que se encarga de la lógica del servidor y la base de datos.

C

Cloud Computing: Uso de servicios de computación a través de internet.

D

DevOps: Conjunto de prácticas que combinan desarrollo y operaciones para mejorar la entrega de software.

E

Endpoint: Punto final de comunicación en una API.

F

Frontend: Parte del desarrollo web que se encarga de la interfaz de usuario.

G

Git: Sistema de control de versiones distribuido.

H

HTTP: Protocolo de transferencia de hipertexto utilizado en la web.

I

IP: Dirección única que identifica un dispositivo en una red.

J

JavaScript: Lenguaje de programación utilizado principalmente en el desarrollo web.

K

Kubernetes: Plataforma de orquestación de contenedores.

L

Linux: Sistema operativo de código abierto.

M

Machine Learning: Rama de la inteligencia artificial que permite a las máquinas aprender de los datos.
